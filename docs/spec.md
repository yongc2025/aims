# AIMS Product Specification

## Overview

AIMS (AI Market Intelligence System) is an AI-driven A-share market intelligence platform.

The system focuses on collecting, validating, storing and visualizing public market information.

## Product Boundary

AIMS is:

- Market data collection system
- Historical market information database
- Intelligence dashboard
- AI-assisted research archive

AIMS is not:

- Trading system
- Investment advisory system
- Price prediction system
