# AIMS Frontend

AI Market Intelligence System frontend.

## Design Principles

- Unified Dark FinTech visual system
- Responsive PC/mobile layouts
- Data-driven components
- No static chart images
- All charts rendered dynamically with ECharts

## Stack

- React
- TypeScript
- Vite
- Ant Design
- Tailwind CSS
- Apache ECharts
- Framer Motion

## Pages

- Dashboard
- Market Data
- Reports
- Analysis
