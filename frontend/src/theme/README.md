# AIMS Global Theme

All pages MUST use the same visual system.

Rules:

- Dark FinTech theme only
- Shared layout components
- Shared color tokens
- Shared typography
- Shared card styles
- Shared chart styles

Pages must not create independent themes.

## Design Tokens

Background:
- Deep navy / black

Primary:
- Technology blue

Positive:
- Green

Negative:
- Red

Warning:
- Amber

## Components

All pages should reuse:

- AppLayout
- DataCard
- ChartPanel
- SectionHeader
- StatusIndicator
